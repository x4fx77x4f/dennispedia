Dennispedia

This site is an unofficial resource for all things in the Dennis Gustafsson cinematic universe that Dennis Gustafsson has worked on. It is not intended to be a replacement for official documentation (i.e. the Teardown modding documentation). The focus is primarily on things that are not already documented, such as undocumented functions, the structure of proprietary binary formats, and (hopefully) exhaustive lists of builds.

You can contribute to this site by making an issue or pull request on GitHub.
